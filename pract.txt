To build a Java Project using Jenkins with parameters to perform some mathematical operations 



Calculator.java
public class Calculator {
    public static void main(String[] args) {
        int a = Integer.parseInt(args[0]);
        int b = Integer.parseInt(args[1]);
        String op = args[2];

        if(op.equals("add")) {
            System.out.println("Result = " + (a + b));
        }
        else if(op.equals("mul")) {
            System.out.println("Result = " + (a * b));
        }
        else {
            System.out.println("Invalid operation");
        }
    }
}

8. 
To create a pipeline script (consisting of stages, parameters etc.) and build a pipeline of jobs in Jenkins


Jenkins pipeline project
This project is parameterized
Name :   DEVICE_OWNER
Default Value: carol
Description: Enter the laptop owner name
Pipeline script
pipeline {
    agent any


    parameters {
        string(
            name: 'DEVICE_OWNER', 
            defaultValue: 'Carol', 
            description: 'Enter the laptop owner name'
        )
    }


    stages {


        stage('stage1') {
            steps {
                echo "Laptop is owned by ${params.DEVICE_OWNER}"
            }
        }


        stage('stage2') {
            steps {
                echo "Hello Carol"
            }
        }


    }
}
Apply save
Build with parameters and check pipeline overview
